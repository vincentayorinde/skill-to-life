# NextSkill

NextSkill is an open-source tech career assessment platform that helps people discover their NextSkill, top career matches, learning resources, salary insights, and entrepreneurship paths.

## Local Development

### Prerequisites

- Node.js 22+
- pnpm
- Git

### Setup

```sh
pnpm install
```

### Commands

```sh
pnpm dev
pnpm dev:web
pnpm dev:api
pnpm build
pnpm test
pnpm lint
pnpm format:check
```
